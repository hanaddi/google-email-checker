How to Install a User Script

Step 1: Install a User Script Manager Extension

You need a user script manager to run custom scripts in your browser. The recommended options are:

  • Tampermonkey (Recommended)
  • Greasemonkey (for Firefox users)
  • Violentmonkey

For Google Chrome, Edge, or Brave:

  1. Open the Chrome Web Store.
  2. Search for Tampermonkey.
  3. Click Add to Chrome â†’ Add Extension.

For Firefox:

  1. Open the Firefox Add-ons Store.
  2. Search for Tampermonkey or Greasemonkey.
  3. Click Add to Firefox â†’ Install.

For Opera:

  1. Open the Opera Add-ons Store.
  2. Search for Tampermonkey.
  3. Click Add to Opera â†’ Install.

For Safari:

  1. Open the Mac App Store.
  2. Search for Tampermonkey.
  3. Click Get â†’ Install.

Step 2: Open the Script URL

  1. After installing the extension, visit: https://hanaddi.github.io/google-email-checker/emailchecker.user.js
  2. Your user script manager (e.g., Tampermonkey) will detect the script and open an installation page.

Step 3: Install the User Script

  1. On the Tampermonkey installation page, review the script details.
  2. Click Install.
  3. The script will now be active and running in the background.

Step 4: Manage and Enable/Disable Scripts

  1. Click on the Tampermonkey icon in the browser toolbar.
  2. Select Dashboard to see installed scripts.
  3. You can enable, disable, or edit scripts as needed.
